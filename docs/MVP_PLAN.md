# Memoirium MVP Plan

## Sprint 1 - Foundation
- Keep the Figma UI as the visual base.
- Add routing, environment config, Supabase client, database schema, and deployment files.

## Sprint 2 - Auth
- Supabase email/password auth.
- Protected dashboard routes.
- Profile creation with username.

## Sprint 3 - Collections
- Create, edit, delete collections.
- Public/private toggle.
- Curator note and mood fields.

## Sprint 4 - Memories
- Create, edit, delete memories.
- Image upload to Supabase Storage.
- Emotion tags.

## Sprint 5 - Public Museum
- /museum/:username public page.
- Exhibition rooms.
- Featured memories.
- Visit tracking.

## Sprint 6 - Showcase Feature
- Memory Constellation with React Three Fiber.
